Catálogo de Padrões de Mapeamento RDF2RDF

Definiu-se um catálogo de padrões de mapeamento que consiste em 11 padrões de mapeamento. Um mapeamento entre termos de ontologia pode ser um mapeamento entre classes ou um mapeamento entre propriedades. Apresenta-se abaixo, uma classificação dos padrões de mapeamento de acordo com o tipo de mapeamento envolvido.

Como existem dois tipos de propriedades (objeto e tipo de dados), agrupou-se os padrões em três grupos: classes, propriedades de objetos e propriedades de tipos dados. Cada grupo foi dividido para resolver um problema específico de mapeamento. Por exemplo, os padrões de mapeamento de classes foram divididos em padrões de mapeamento entre classes semanticamente equivalentes e padrões de mapeamento entre classes não equivalentes (determinar quando duas classes são semanticamente equivalentes ou não, esta fora do escopo deste trabalho, mas o leitor pode ver [Rodriguez e Egenhofer, 2003] para mais detalhes sobre este assunto).

Observação: Para visualizar o padrão, basta clicar no nome do padrão.
