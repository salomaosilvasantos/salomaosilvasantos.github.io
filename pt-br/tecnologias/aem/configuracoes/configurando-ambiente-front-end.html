Configurando o Ambiente de Desenvolvimento Local para Front-end

Requesitos:
    JDK 11 Apache Maven Visual Studio

Instalação JDK11 (Java Development Kit)

Para instalar o JDK no Windows é necessário realizar o download do arquivo de instalação. Para isso deve-se acessar o site da Oracle e baixar versão do JDK correspondente ao sistema operacional e arquitetura (32 ou 64 bits) utilizada.
Após o download do arquivo de instalação ter sido concluído, a instalação do JDK é feita executando-se o programa de instalação. O nome do arquivo de instalação para o Windows tem o formato jdk--windows-.exe, onde é a versão do JDK e arquitetura pode ser 32 ou 64 bits, por exemplo, jdk-11.0.7-windows-x64_bin.exe.
Para instalar o JDK no Windows, executar o programa de instalação e clicar no botão Next em todas as telas apresentadas. O processo é simples e rápido.

Instalação Apache Maven

O principal requisito para instalar e usar o apache maven na sua máquina é ter previamente instalado o JDK (Java Development Kit). Para instalar Apache Maven no Windows é necessário realizar o download do arquivo de instalação. Para isso deve-se acessar o site Apache Maven Project e baixar o arquivo apache-maven-X.X.X-bin.zip que é a versão que contém o executável do Apache Maven.
Após terminar o download do Apache Maven, basta extrair o conteúdo para a pasta desejada. No caso deste tutorial utilizará-se o Disco Local (C:). Concluindo a extração dos arquivos terá se o resultado da figura 02.
Para instalar o JDK no Windows, executar o programa de instalação e clicar no botão Next em todas as telas apresentadas. O processo é simples e rápido.
